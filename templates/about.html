关于「NBA 新闻聚合」

「NBA 新闻聚合」是清华大学计算机系 2019 年小学期 Python 爬虫大作业项目，全部代码在 GitHub 上开源。

采用的技术：

  • Scrapy：实现新闻、球队和球员信息的爬取；
  • Django：驱动服务器；
  • jieba：用于处理爬取到的语料库；
  • MySQL：后台数据库；
  • Scrapyd：Scrapy 守护进程，用于后台数据爬取。
